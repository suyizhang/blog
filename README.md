2. 指定数据目录启动：`mongod --dbpath <数据目录路径>`
mongod --dbpath data/db

停止MongoDB：
1. 在运行MongoDB的终端按 Ctrl+C
2. 或者使用`mongosh` 连接到MongoDB后执行`db.shutdownServer()`
验证MongoDB状态：

1. 使用`mongosh` 命令连接数据库
2. 检查27017端口是否在监听：`lsof -i :27017`
